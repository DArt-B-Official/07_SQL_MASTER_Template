<h1 align="center">SQL MASTER Template</h1>

**SQL MASTER 정규과제 템플릿**입니다.  
이 템플릿은 다트비 7기에서 매주 주어지는 SQL MASTER 정규 과제를 정리하고 제출하기 위한 기본 템플릿을 제공합니다.  

---

## 학습 목표
- **데이터 분석을 위한 SQL의 역할과 구조를 이해**하고, 다양한 데이터 환경에서 활용되는 SQL의 사고방식을 익힙니다.
- **문자열·날짜·집계·윈도우 함수 등 SQL 데이터 가공 기법**을 활용하여 데이터를 목적에 맞게 정제하고 변환합니다.
- **시계열 분석, 사용자 분석, 매출 분석 등 실무 분석 패턴**을 SQL로 구현하며 데이터 기반 의사결정 역량을 기릅니다.
- **여러 테이블을 조합하고 지표를 설계**하여 실제 서비스 데이터 분석에 적용할 수 있는 쿼리 작성 능력을 함양합니다.
- **다양한 SQL 환경**에서 문법을 유연하게 적용하고, 실습을 통해 문제 해결 능력을 강화합니다.

---

## 📂 제출 방법 (리마인드)
1. 이 템플릿 레포지토리의 `assignments` 폴더 내의 마크다운 파일을 개인 계정 레포지토리에 복사합니다.  
2. 매주차 과제를 `weekX/README.md`에 정리합니다. (형식은 자유)  
3. 과제가 완료되면 **개인 레포 링크**를 과제 제출 시트에 업로드합니다.  
> (이미지 폴더는 개인 레포에 따로 만들어서 저장을 해야 업로드가 됩니다.)
